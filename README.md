# QuickBus

QuickBus is an online bus ticket booking platform that allows users to search buses, view seat availability, and book tickets quickly and efficiently. The project provides a user-friendly interface with agent and operator dashboards to manage bookings.

---

## Features

- User registration and login
- Search buses by source and destination
- View available seats and select seats
- Book tickets online
- Agent login and dashboard
- Operator login and dashboard
- Booking history and reports

---

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Spring Boot (Java)  
- **Database:** MySQL  
- **IDE:**  Eclipse EE
